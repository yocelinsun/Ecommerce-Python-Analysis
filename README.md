# 电商用户行为与订单数据分析 Ecommerce-Python-Analysis
Python 数据分析实战项目｜用户画像｜行为转化漏斗｜销售分析｜自动化Excel报表

## 项目介绍
本项目基于真实电商业务数据，使用 Python 进行全流程数据分析，
包括SQL取数、数据清洗、用户画像构建、行为漏斗分析、销售趋势分析，并自动生成结构化 Excel 报表。

## 技术栈
- Python
- Pandas / Matplotlib / Seaborn
- MySQL 数据读取
- 自动化 Excel 报表生成

## 分析内容
1. 整体业务指标统计
2. 用户画像分析（性别、年龄、会员等级）
3. 用户行为转化漏斗
4. 每日销售额趋势分析
5. 订单金额分布分析

## 项目产出
- 完整数据分析代码
- 多维度可视化图表
- 自动化生成 Excel 分析报表

### 数据来源
https://tianchi.aliyun.com/dataset/222889
  -选取order.csv, order_item.csv, user.csv, user_behavior.csv
